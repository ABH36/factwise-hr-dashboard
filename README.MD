Follow these steps to run the project locally.

### Prerequisites
- Node.js (v16+)

### 1. Backend Setup (Server & Database)
The backend handles API logic and database seeding.

```bash
# Navigate to backend

# Install dependencies

# Set up Environment Variables

# Start the Server
npm run dev
